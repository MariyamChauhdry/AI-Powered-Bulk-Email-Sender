# AI-Powered Bulk Email Sender

An AI-powered Python tool that lets you draft and send personalized bulk emails using **Google Gemini**, with built-in support for file attachments, email tracking, and logging via **Firebase Firestore**. Designed with a clean and simple UI for easy use ‚Äî perfect for newsletters, campaigns, or internal communication.

---

## Features

- **AI-generated Emails** using Google Gemini API
- **Upload Excel** list of email recipients
- **Attach Files** like PDFs, images, or docs
- **Send via Free Email API** (e.g., Brevo, Mailgun, SMTP)
- **Firebase Firestore Logging**
- **Tracking of Emails**
- Easy-to-use GUI

## How It Works

1. Upload a `excel` file with email addresses.
2. Enter a prompt for Gemini AI (e.g., ‚ÄúDraft a newsletter for our new product launch‚Äù).
3. Optionally attach a file.
4. Click **Send** ‚Äî emails are generated, sent, and logged.
5. View logs in Firebase Firestore.
